# internship-tracker
A full stack application that lets users track their internship progress built with React, Node.js, Express and PostgreSQL

# Planned features
- User authentication
- Internship application dashboard
- Add/Edit/Delete Internships 
- Status tracking
- Deadline reminders
- Resuem version tracking 
- Search and filtering 
- Analytics dashboard
- Email Integration 
- Browser extension 

## Tech Stack

Frontend:
- React

Backend:
- Node.js
- Express

Database:
- PostgreSQL

Authentication:
- JWT

Deployment:
- Vercel
- Render
- Neon
